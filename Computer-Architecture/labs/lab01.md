# 实验一

## 实验信息

| 项目 | 内容 |
| --- | --- |
| 实验题目 | 实验一 MIPS指令体系结构和MIPS指令 |
| 课程 |  |
| 专业 | 2022级 |
| 实验时间 | 2024年10月21/22/25日 |
| 实验地点 |  |
| 实验环境 | 指令级模拟器 |

## 实验目的

1. 了解和熟悉指令级模拟器
2. 熟练掌握MIPSsim模拟器的操作和使用方法
3. 熟悉MIPS指令系统及其特点，加深对MIPS指令操作语义的理解
4. 熟悉MIPS体系结构

## 实验内容

1. 分析MIPS基本指令的执行流程，在模拟器上验证其对应的微指令
2. 以alltest.s为例，写出该程序中对应的寄存器、内存、PC等值的内容
3. 分析每条指令执行后变化的原因

## 实验步骤

1. 启动MIPSsim模拟器
2. 载入样例程序alltest.s
3. 单步执行程序，观察每条指令执行后寄存器、内存、PC等值的变化
4. 完成《实验一 alltest.s 内容分析》Excel文件
5. 记录并分析每条指令执行后变化的原因
