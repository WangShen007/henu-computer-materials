# 实验2: 密码显示（二选一）
## 实验内容
编写WinForms 应用程序，当用户在密码框中输入一个密码字符时，在文本框中要立即将输入的密码字符显示出来。
## 实验目标
学习WinForms控件：了解WinForms中的控件基本用法。

事件处理：理解如何在WinForms应用程序中如何以事件的方式处理用户输入。

登录窗体（二选一）
## 实验内容
编写WinForms 应用程序，制作登录窗体，包含登录、退出按钮，用户名文本框、密码输入框。设计主窗体，包含MenuStrip菜单项，并注册菜单项事件。

点击登录，进行用户名、密码验证，如果输入错误3次，退出应用程序，如果登录成功显示主窗体。点击退出，退出应用程序。
## 实验目标
学习WinForms控件：了解WinForms中的控件基本用法。

事件处理：理解如何在WinForms应用程序中如何以事件的方式处理用户输入。
