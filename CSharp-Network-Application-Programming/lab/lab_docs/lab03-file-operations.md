# 实验3: 文件拷贝
## 实验内容
（1）设计文件复制界面，增加文本文件拷贝和通用文件拷贝按钮。

（2）文本文件拷贝：

利用OpenFileDailog组件，构造文件选择对话框，选择待复制文本文件。注意限制文件后缀为\*.txt。利用File类读取文件，并将文件内容写入到新文件中。新文件名格式为旧文件名\_复制.旧文件名后缀。

（3）通用文件拷贝：

利用OpenFileDailog组件，构造文件选择对话框，选择待复制文本文件。利用文件流，实现通用文件拷贝。通用文件拷贝功能，不限制文件类型。
## 实验目标
综上所述，掌握OpenFileDailog组件、File类、FileStream类、字符串格式化的方法，同时提升编程能力、问题解决能力和数学素养。
