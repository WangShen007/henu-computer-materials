# 计算机网络期末重点

## 考试重点提示

1. **第四、三、五章是重点章节**

## 第一章 概述

1. 计算机网络的概念和分类

5. 协议的概念、协议和服务的关系

## 第二章 物理层

1. 物理层接口的基本特性
2. **香农公式**：信道的极限信息传输速率与带宽及信噪比的关系
   - C = W × log₂(1 + S/N)
3. **奈氏准则**：码元传输的最高速率，最大信息传输速率
4. 通信的三种基本方式（单向通信、双向交替通信、双向同时通信）
5. 信道复用技术的种类及其主要特征（FDM、TDM、WDM、CDMA等）
6. CDMA的简单应用

## 第三章 数据链路层（重点）

1. 数据链路层的两种信道、三个基本问题
2. 点到点协议PPP及其透明传输的实现（同步传输）
3. **差错检测技术**，CRC的冗余码计算
4. 以太网的标准、拓扑类型、传输介质等
5. **CSMA/CD协议**的工作过程，及争用期、最短帧长等概念和应用
   - 争用期 = 2τ（端到端往返传播时延）
   - 最短帧长 = 2τ × 数据传输速率
6. 以太网的MAC地址、帧格式
7. 以太网交换机的特点，如带宽、自学习算法、转发算法

## 实验相关重点

### 基本操作命令

- **VLAN划分**：`vlan 100` → `port GigabitEthernet 0/0/1 to 0/0/8`
- **配置端口IP**：`interface vlan 1` → `ip address 192.168.1.11 255.255.255.0`
- **Trunk配置**：`port link-type trunk` → `port trunk allow-pass vlan all`
- **静态路由**：`ip route-static 目的网络 掩码 下一跳`
- **RIP配置**：`rip` → `version 2` → `network 网络号`
- **OSPF配置**：`ospf 100 router-id x.x.x.x` → `area 0` → `network 网络号 掩码`
- **查看路由表**：`display ip routing-table`
- **查看VLAN**：`display vlan`
