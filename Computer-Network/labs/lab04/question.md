# 实验4：网络互连与路由器配置

## 实验目的

1. 理解网络互连的概念和路由器的工作原理，掌握路由器的功能实现及配置
2. 理解路由表、掌握静态路由的配置
3. 掌握动态路由的配置方法，理解RIP-2协议的工作过程
4. 理解链路状态路由协议的工作过程
5. 理解PPP、PPPoE协议，掌握路由器上相关协议的配置

## 实验内容

1. 路由器的基本配置
2. 路由协议的配置
3. PPP协议配置
4. PPPoE配置

## 实验环境

- AR2200E路由器，2台
- PC机，3台
- 网络跳线（交叉线或平行线），3根
- 特殊AS接口网络跳线，1根
- SecureCRT等工具软件
