# 实验1：网络跳线的制作

## 实验目的

1. 认识RJ-45水晶头，学习掌握RJ45水晶的制作工艺及操作规程，培养熟练制作各种跳线的能力及对各种跳线的选择使用的能力
2. 学习测线器的各端口及指示灯的功能，培养正确使用测线器对UTP双绞线跳线进行通断及线序测试的能力
3. 了解网络布线的线缆、配件及工具等

## 实验内容

1. 学习UTP线缆制作工具剥线器、压线钳的使用方法
2. 制作568A/568B直通线、交叉线、控制线（反转线）
3. 使用测线器测试所制作的网络跳线连通性
4. 参观综合布线系统展示柜

## 实验环境

- 5类UTP线缆
- 8芯RJ-45水晶头
- 剥线器
- 压线钳
- 测线器
