### 题目1：什么叫指令？什么叫指令系统？

> * **指令**：指令是计算机执行某种操作的命令，也就是常说的机器指令。
> * **指令系统**：一台机器中所有机器指令的集合，称这台计算机的指令系统。

## 四、 应用题 / 计算题

### 题目1：设某机主频为8MHz，每个机器周期平均含2 个时钟周期，每条指令平均有2.5 个机器周期，试问该机的平均指令执行速度为多少MIPS？若机器主频不变，但每个机器周期平均含4 个时钟周期，每条指令平均有5 个机器周期，则该机的平均指令执行速度又是多少MIPS？由此可得出什么结论？

**解题步骤：**

1. **计算第一种情况下的执行速度**：
* 时钟周期 = $\frac{1}{8\text{MHz}} = 0.125 \times 10^{-6}\text{s} = 125\text{ns}$
* 机器周期 = $125\text{ns} \times 2 = 250\text{ns}$
* 平均指令周期 = $250\text{ns} \times 2.5 = 625\text{ns}$
* 平均指令执行速度 = $\frac{1}{625\text{ns}} = 1.6\text{MIPS}$


2. **计算参数改变后的执行速度**：
* 机器周期 = $125\text{ns} \times 4 = 500\text{ns} = 0.5\mu\text{s}$
* 平均指令周期 = $0.5\mu\text{s} \times 5 = 2.5\mu\text{s}$
* 平均指令执行速度 = $\frac{1}{2.5\mu\text{s}} = 0.4\text{MIPS}$


3. **得出结论**：
两个主频相同的机器，执行速度不一定一样。

### 题目3：设机器数字长为8 位（含一位符号位），若A = +15，B = +24，求$[A+B]*\text{补}$ 和$[A-B]*\text{补}$ 并还原成真值。

**解题步骤：**

1. **求出各数的补码形式**：
* $A = +15 = +0001111 \implies [A]_\text{补} = 0,0001111$
* $B = +24 = +0011000 \implies [B]_\text{补} = 0,0011000$
* $[-B]_\text{补} = 1,1101000$


2. **计算 $[A+B]_\text{补}$**：

$$[A+B]_\text{补} = [A]_\text{补} + [B]_\text{补} = 0,0001111 + 0,0011000 = 0,0100111$$


* 还原真值：$A+B = +0100111 = +39$


3. **计算 $[A-B]_\text{补}$**：

$$[A-B]_\text{补} = [A]_\text{补} + [-B]_\text{补} = 0,0001111 + 1,1101000 = 1,1110111$$


* 还原真值：$A-B = -0001001 = -9$
