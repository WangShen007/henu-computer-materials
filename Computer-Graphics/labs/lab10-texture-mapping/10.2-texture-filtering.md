给球体添加纹理映射。

#### 1. 设置控制纹理映射
设置纹理映射控制的函数，参数分别如下：
```c
    glTexParameteri(GL_TEXTURE_1D, GL_TEXTURE_WRAP_S, GL_REPEAT);   
    glTexParameteri(GL_TEXTURE_1D, GL_TEXTURE_MAG_FILTER, GL_LINEAR);    
    glTexParameteri(GL_TEXTURE_1D, GL_TEXTURE_MIN_FILTER, GL_LINEAR);   
```

#### 2. 定义一纹理贴图
定义一维纹理贴图函数，参数分别如下：
```c
    glTexImage1D(GL_TEXTURE_1D, 0, 4, stripeImageWidth, 0, GL_RGBA, GL_UNSIGNED_BYTE, stripeImage);
```
<details>

<pre><code><font size=4.6>
一维纹理：
创建一个简单的一维纹理可使用 glTexImage1D() 函数。

void glTexImage1D(GLenum target, GLint level, GLint internalFormat, GLsizei width, GLint border, GLenum format, GLenum type, const GLvoid *texels);

该函数用来定义一个一维纹理。除了参数 texels 现在为一维数组之外，所有参数和 glTexImage2D() 函数中相应的参数有相同的含义。
</font>
</code></pre>
</details>

#### 3. 设置纹理映射方式
设置纹理映射方式，函数参数如下：
```c
    glTexEnvf(GL_TEXTURE_ENV, GL_TEXTURE_ENV_MODE, GL_MODULATE);  
```

#### 4. 设置纹理坐标自动生成函数
设置纹理坐标自动生成函数参数如下：
```c
    glTexGeni(GL_S, GL_TEXTURE_GEN_MODE, GL_OBJECT_LINEAR);       
    glTexGenfv(GL_S, currentPlane, currentCoeff);
```

<pre><code><font size=4.6>
自动纹理坐标生成函数：
下面原型函数可提供自动纹理坐标生成功能：

void glTexGen{ifd}(GLenum coord, GLenum pname, TYPE param);

参数说明如下：
（1）coord 指定要产生的纹理坐标类型，可取 GL_S、GL_T、GL_R 或者 GL_Q；
（2）pname 指定一个纹理坐标生成函数的符号名，必须是 GL_TEXTURE_GEN_MODE、GL_OBJECT_PLANE、GL_EYE_PLANE 之一；
（3）param 指定一个指向纹理生成参数数组的指针。使用 glEnable/glDisable 函数通过使用 GL_TEXTURE_GEN_S、GL_TEXTURE_GEN_T、GL_TEXTURE_GEN_R、或 GL_TEXTURE_GEN_Q 来启用或关闭自动纹理生成功能。
</font>
</code></pre>
</details>
