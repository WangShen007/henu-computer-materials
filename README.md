# 河南大学计算机资料 (HENU Computer Science Materials)

本项目汇总了河南大学计算机科学与技术专业各门课程的资料，包括试卷、习题、知识点总结等。

资料来源于 [Henu-Kaguya](https://github.com/Henu-Kaguya/Henu-Kaguya) 组织下的各科仓库。

## 📚 包含科目

| 目录 | 内容 |
|------|------|
| Computer-Network | 计算机网络 |
| Digital-Image-Processing | 数字图像处理 |
| Computer-Graphics | 计算机图形学 |
| Computer-Architecture | 计算机体系结构 |
| Software-Engineering | 软件工程 |
| Operating-System | 操作系统 |
| Machine-Learning | 机器学习 |
| Compiler-Principles | 编译原理 |
| CSharp-Programing | C# 程序设计 |
| Principles-of-Computer-Organization | 计算机组成原理 |
| Database-Principles | 数据库原理 |
| CSharp-Programing-Demo | C# 程序设计 Demo |
| CSharp-Network-Demo | C# 网络编程 Demo |
| Operational-Research | 运筹学 |
| CSharp-Network-Application-Programming | C# 网络应用编程 |
| C-Primer-Plus-Programming | C Primer Plus 编程 |
